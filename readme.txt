Installation and Configuration Steps for Laravel Framework
1. First of all, install xampp with latest php version and run xampp
2. Download and install composer
3. Create Laravel project using CLI(composer create-project --prefer-dist laravel/laravel igctest)
4. Create database in localhost (igc_test)
5. Configure the database in laravel project environment
6. Create migration using CLI to creat tables categories and news
7. Add required columns in migration file
8. Migrate files to create tables in database
9. Create Model(NewsModel) and Controller(News) using CLI 
10. Manage routes fron web.php to load desired controller method and load view file
